## рҹҡҖ What is CMDR SYS?

**CMDR SYS** is your personal mission log, intel tracker, and notepad вҖ” built for Elite Dangerous Commanders who need to keep track of the galaxy without tabbing out of the cockpit. Whether you're hunting bounties, plotting trade routes, or tracking faction reputation, CMDR SYS keeps your notes organized and accessible.

> *"In a galaxy of 400 billion star systems, you're going to need a notepad."*

## рҹ“Ұ Installation

### рҹ–ҘпёҸ Desktop (Electron)

```bash
# Clone the repository
git clone https://github.com/SgtEpsilon/cmdrsys.git

# Navigate to the Electron directory
cd cmdrsys/Electron

# Install dependencies
npm install

# Run the app
npm start
```

### рҹ“ұ Android

```bash
# Navigate to the Android directory
cd cmdrsys/Android

# Build and run (see android build and run.txt for full instructions)
```
